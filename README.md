# 🚀 SuperCRM Telegram Bot

CRM-бот для управління клієнтами, товарами на складі та виставлення рахунків ФОП.

---

## 📋 Функціонал

### Для клієнтів
- Реєстрація з прийняттям умов
- Каталог товарів по категоріях (фото, ціна, наявність)
- Кошик з управлінням кількістю
- Оформлення замовлення з реквізитами ФОП для оплати
- Перегляд своїх замовлень та статусів
- Запит рахунку на оплату (отримує PDF)
- Налаштування доставки Новою Поштою

### Для адміна
- Підтвердження нових клієнтів
- Прив'язка клієнтів до компаній
- Управління товарами (додавання, фото, ціни, залишки)
- Управління брендами та категоріями
- Перегляд та фільтрація замовлень
- Зміна статусів замовлень (клієнт отримує push-сповіщення)
- Виставлення PDF-рахунків (відправляється клієнту автоматично)
- Налаштування реквізитів ФОП

## ⚙️ Встановлення та запуск

### 1. Отримайте токен бота
1. Відкрийте Telegram, знайдіть [@BotFather](https://t.me/BotFather)
2. Відправте `/newbot`
3. Вкажіть назву: `SuperCRM`
4. Вкажіть username: `your_supercrm_bot` (придумайте унікальний)
5. Скопіюйте токен вигляду `123456789:AAF...`

### 2. Дізнайтесь свій Telegram ID
Напишіть боту [@userinfobot](https://t.me/userinfobot) — він поверне ваш ID (число).

### 3. Налаштуйте .env файл
```bash
cp .env.example .env
```
Відкрийте `.env` та вставте:
```
BOT_TOKEN=ВАШ_ТОКЕН_ТУТ
ADMIN_IDS=ВАШ_TELEGRAM_ID
```
Якщо адмінів кілька: `ADMIN_IDS=123456789,987654321`

### 4. Запуск локально (для тесту)
```bash
# Встановіть Python 3.11+
pip install -r requirements.txt
python bot.py
```

## 🌐 Розгортання на Render.com (безкоштовно)

### Крок 1 — GitHub
1. Зареєструйтесь на [github.com](https://github.com)
2. Натисніть **New repository** → назва `supercrm-bot` → **Create**
3. Завантажте всі файли проєкту (кнопка **uploading an existing file**)

### Крок 2 — Render
1. Зареєструйтесь на [render.com](https://render.com) через GitHub акаунт
2. Натисніть **New +** → **Web Service**
3. Підключіть репозиторій `supercrm-bot`
4. Налаштування:
   - **Name**: supercrm-bot
   - **Runtime**: Python 3
   - **Build Command**: `pip install -r requirements.txt`
   - **Start Command**: `python bot.py`
5. У розділі **Environment Variables** додайте:
   - `BOT_TOKEN` = ваш токен
   - `ADMIN_IDS` = ваш Telegram ID
6. Натисніть **Create Web Service**

✅ Бот запуститься автоматично і буде працювати 24/7!

## 📁 Структура проєкту

```
supercrm_bot/
├── bot.py                    # Головний файл запуску
├── requirements.txt
├── .env.example
├── models/
│   └── database.py           # SQLite схема та підключення
├── handlers/
│   ├── register.py           # Реєстрація клієнтів
│   ├── catalog.py            # Каталог товарів
│   ├── cart.py               # Кошик та оформлення замовлення
│   ├── orders_client.py      # Замовлення клієнта
│   ├── profile.py            # Профіль та Нова Пошта
│   ├── admin_orders.py       # Управління замовленнями (адмін)
│   ├── admin_products.py     # Управління товарами (адмін)
│   ├── admin_clients.py      # Управління клієнтами (адмін)
│   ├── admin_settings.py     # Налаштування, ФОП, компанії (адмін)
│   └── admin_invoice.py      # Генерація PDF рахунків (адмін)
├── keyboards/
│   ├── client_kb.py          # Клавіатури для клієнтів
│   └── admin_kb.py           # Клавіатури для адміна
└── utils/
    ├── states.py             # FSM стани форм
    ├── helpers.py            # Допоміжні функції
    └── pdf_generator.py      # Генерація PDF рахунків
```

---

## 🔧 Перші кроки після запуску

1. Напишіть боту `/start` — ви зайдете як адмін
2. Перейдіть у **⚙️ Налаштування** → **💳 Реквізити ФОП** — введіть дані
3. Додайте категорії та бренди
4. Додайте перші товари через **📦 Товари**
5. Поділіться посиланням на бота з клієнтами
